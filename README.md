# Cat Tracker — Home Assistant add-on

This repository contains **only** the Supervisor add-on (folder **`cattracker/`**) plus **`repository.json`** for the Home Assistant add-on store.

Add **`https://github.com/1313picklw1/Cat-Tracker-HAOS`** under **Settings → Add-ons → Repositories**, then install **Cat Tracker**.

See **`cattracker/README.md`** for configuration, MQTT, and disk-space notes.
